# Currículo HTML

Projeto simples de **currículo desenvolvido em HTML** com o objetivo de apresentar informações profissionais básicas.

## Sobre o projeto

Este projeto consiste em uma página HTML contendo um currículo com informações pessoais, formação acadêmica, habilidades e cursos complementares.

O objetivo é praticar **estruturação de páginas utilizando HTML** e organização de informações profissionais.

## Estrutura do projeto

* `index.html` — página principal contendo o currículo.

## Conteúdo do currículo

O currículo apresenta:

* Informações pessoais
* Descrição profissional
* Formação acadêmica
* Soft skills
* Cursos e aprimoramentos

## Como visualizar

1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` em qualquer navegador

ou acesse diretamente pelo GitHub Pages (se estiver publicado).
